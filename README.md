# htmx-fullstack-project
 In this project I will be using express.js, htmx and bootstrap to create an interesting web application.

Project was built using Node version:
```bash
Node.js v21.5.0
```

```bash
npm install
cd ./frontend
npm install
cd ../backend
npm install
```

To run the project

```bash
npm run backend
```

```bash
npm run frontend
```
